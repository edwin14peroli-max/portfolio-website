# portfolio-website
Simple project 1

**Portfolio Website**

A simple and responsive personal portfolio website built using HTML and CSS. This project showcases basic web development skills, including page structure, styling, navigation, and responsive design principles.

**Features**
Home page with personal introduction and portfolio overview.
Contact page for visitors to get in touch.
Clean and user-friendly interface.
Built using pure HTML and CSS without any frameworks.

**Technologies Used**
* HTML5
* CSS3

**Pages**
Home Page – Displays personal information, skills, and portfolio content.
Contact Page – Provides contact details and a contact section for communication.


**Purpose**
This project was created to practice and demonstrate fundamental front-end web development skills using HTML and CSS.
